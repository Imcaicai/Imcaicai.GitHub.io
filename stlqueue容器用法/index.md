# 【STL】queue容器用法


## 1 创建queue

```c++
queue<int> q;
```

## 2 常用操作

| front()       | 返回 queue 中第一个元素的引用。        |
| ------------- | -------------------------------------- |
| back()        | 返回 queue 中最后一个元素的引用        |
| push(T&& obj) | 以移动的方式在 queue 的尾部添加元素。  |
| pop()         | 删除 queue 中的第一个元素。            |
| empty()       | 如果 queue 中没有元素的话，返回 true。 |
| size()        | 返回 queue 中元素的个数。              |
